Article App using home24 public API where users should be able to "like" and "dislike" articles

## Getting Started

Clone project to your local system directory using your terminal

### Tool Required

* Android studio
* Kotlin plugin
* Gradle 3.1.3+
* Adb driver
* Android Device or Emulator for testing
